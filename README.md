# 📚 API de Controle de Estudos

Este é meu primeiro projeto de backend desenvolvido com Node.js.  
O objetivo deste projeto é criar um sistema simples para registrar sessões de estudo e matérias.

## 💡 Ideia

A ideia por trás desse mini projeto é construir um sistema de controle de estudos onde o usuário pode:

- Registrar as matérias que está estudando
- Registrar sessões de estudo
- Acompanhar o progresso dos estudos
- Organizar sua rotina de aprendizagem

Este projeto serve como um projeto de prática para entender como funciona o desenvolvimento backend utilizando Node.js.

## 🎯 Objetivo

Os principais objetivos deste projeto são:

- Praticar desenvolvimento backend
- Aprender a criar uma API REST
- Entender como servidores funcionam com Node.js
- Praticar a organização de um projeto backend
- Aprender a lidar com requisições e respostas

## 🛠 Stack

Este projeto utiliza:

- Node.js
- Express
- CORS
- JavaScript

## 📦 Funcionalidades (Planejadas)

- Adicionar uma matéria de estudo
- Listar matérias de estudo
- Registrar sessões de estudo
- Acompanhar o progresso dos estudos

## 🚀 Propósito de Aprendizado

Este projeto faz parte da minha jornada de aprendizado em desenvolvimento backend e no aprimoramento das minhas habilidades em programação.
